# Password Strength Meter & Breached Credential Checker

A full-stack MERN web application that provides real-time password strength
analysis and checks whether a password has been exposed in known data breaches.

## Features

- Real-time password strength analysis using zxcvbn
- Breached credential checking via Have I Been Pwned API (k-Anonymity)
- User authentication with JWT
- Check history for registered users
- Fully responsive — mobile and desktop

## Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Authentication:** JWT + bcrypt
- **Deployment:** Vercel (frontend) + Render (backend)
